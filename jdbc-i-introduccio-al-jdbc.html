JDBC I. Introducció al JDBC

Entrada ubicada a la categoria bd amb els tags: PostGreSQL java JDBC

Introdució

Amb aquests continguts treballarem la manera de comunicar els nostres programes orientats a objecte amb les nostres bases de dades objecte relacionals.

Les tecnologies concretes que farem servir són el JDK 7 i PostGreSQL 9.4 [1]. Els continguts pressuposen que tens prou nocions d'aquestes dues tecnologies.

Els continguts es poden dividir en tres parts:

  • una primera part d'introducció al JDBC (què és, instal·lació i primeres nocions d'ús)

    S'aprofitarà aquesta part per introduir una nova eina pels nostres desenvolupaments: el Apache Maven.

  • una segona part en la que analitzarem, a partir d'exemples, el funcionament del JDBC amb un model purament relacional.

  • finalment, a la tercera part d'aquesta introducció analitzarem el funcionament amb les diferents característiques objecte-relacional que ens ofereix PostGreSQL.

En el títol d'alguns dels exercicis d'aquest tema, apareix una marca (★). Aquesta estrella indica que l'exercici és d'ampliació o repàs. Els exercicis sense marca estan dissenyats per què disposis d'una referència bàsica de que has entès els continguts.

Notes

[1]Altres versions funcionaran molt probablement sense grans canvis.
