Ritual Hola Món!

Entrada ubicada a la categoria poo amb el tag: java

Introducció

Aquest document guia l'estudiant a l'edició i execució del seu primer programa en java.

Per a poder-lo seguir, cal disposar de:

  • jdk
  • editor de text

Per simplicitat, la guia pressuposa que el sistema operatiu és Ubuntu 12.04, el JDK és el JDK d'Oracle en versió 7, i l'editor el gedit. És perfectament possible seguir aquesta guia amb una configuració diferent. En aquest cas, queda com exercici, l'adaptació de les passes al sistema escollit.

En cas que no es disposi del JDK, es pot considerar la guia d'instal·lació del jdk

Continguts

Passem a l'acció! Segueix les passes:

  1. obre un terminal (ex. <ctrl><alt>t)

  2. crea una carpeta on treballar i situat dins:

    $ mkdir ~/programacio
$ cd ~/programacio

  3. edita un fitxer anomenat HolaMon.java (atenció a les majúscules)

    $ gedit HolaMon.java &

  4. escriu al fitxer el següent contingut:

    1

     /*
  * Programa que saluda al món
  */
 public class HolaMon {
     public static void main(String[] args) {
         System.out.println("Hola Món!");
     }
 }

  5. guarda el document (ex. <ctrl>s), torna al terminal i compila escriu:

$ javac HolaMon.java
  1. si apareix algun error, revisa que el text que has introduït al fitxer contingui exactament l'indicat. Corregeix el que calgui i torna a compilar fins que no s'indiqui cap error.
  2. executa el teu programa
$ java HolaMon

Felicitats! Acabes de realitzar el teu primer programa en Java!

Exercicis

Exercici 1: Preguntes sobre el ritual

Contesta les següents preguntes. Potser et caldrà fer proves amb el codi de les dues versions:

  1. Perquè creus que pot ser tan habitual que es faci un programa tipus "hola món!" quan entres a un nou llenguatge?
  2. Què creus que contenir l'argument args? Com es feia això en C? Què passa si intentem fer un main() sense arguments com podíem fer a C?
  3. Llença una hipòtesi de per què pot ser que no calgui fer quelcom equivalent al #include <stdio.h> de C per poder escriure un missatge per pantalla.
