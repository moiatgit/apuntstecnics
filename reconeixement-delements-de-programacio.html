Reconeixement d'elements de programació

Entrada ubicada a la categoria poo amb el tag: java

Introducció

Aquest document serveix d'orientació a l'estudiant del llenguatge de programació Java per endinsar-se en la seva sintaxi tot seguint els tutorials d'ajuda proporcionats pels propis dissenyadors del llenguatge.

Està format per tres exercicis que requereixen la lectura de diferents documents per a poder respondre a una sèrie de qüestions.

Els documents referenciats apareixen en anglès. Es pressuposa que l'estudiant disposa del coneixement suficient d'aquesta llengua per realitzar els exercicis. Altrament, es recorda l'existència de traductors automàtics. És d'esperar que aquestes eines, no sent ni de lluny perfectes, esperonin l'estudiant a obtenir el necessari domini de la lingua franca entre els desenvolupadors d'aplicacions. [1]

Exercicis

Exercici 1. Tipus primitius del Java

Llegeix les pàgines:

  • Primitive Data Types
  • Arrays

A continuació, contesta les següents preguntes:

  1. Fes una taula que relacioni els tipus primitius de que disposa el Java. Inclou una petita descripció, el rang de valors i el valor per defecte. Indica, si existeix, el corresponent amb el del llenguatge C.

  2. descriu amb les teves paraules el significat de literal. Inclou exemples teus.

  3. descriu amb les teves paraules com es declaren i s'accedeixen els arrays. Compara'ls amb el que coneixies de C. A la teva comparació inclou avantatges/inconvenients que trobes.

  4. De les següents declaracions, indica si són vàlides i si són recomanables, i en cas que no, el perquè

    int[] notes;
int notes[];
int notes[100];
int[10] notes;

  5. Quina diferència hi ha entre declarar una variable de tipus array i crear un array? Com anava això en C?

  6. Per a què serveix arraycopy()? Indica almenys dues raons per les quals és preferible fer servir aquesta funció en comptes de fer la còpia amb un for()

  7. Fes un programa que:

    1. crei un vector amb els primers cinc anys a partir del del teu naixement
    2. fent servir arraycopy() copii del segon al quart any en un altre vector
    3. mostri els anys del segon vector en ordre invers.

Exercici 2. Operadors del Java

Llegeix les pàgines:

  • Summary of Operators
  • Expressions, Statements, and Blocks

A continuació, contesta les següents preguntes:

  1. indica els operadors que ofereix el Java en format de taula. Afegeix una descripció amb les teves paraules i compara'ls amb els que coneixies de C.
  2. descriu l'ús recomanat dels parèntesis a les expressions de Java
  3. descriu què és un statement. Prova de trobar una traducció al teu idioma.
  4. descriu què és un bloc. Compara aquest concepte amb el que coneixes de C.

Exercici 3. Control de flux d'execució

Llegeix les pàgines de la secció següent:

  • Control Flow Statements

A continuació, contesta les següents preguntes:

  1. descriu el funcionament de les estructures condicionals del java (if, switch) Compara-les amb les del llenguatge C
  2. descriu el funcionament de les estructures mentre (while) Compara-les amb les del llenguatge C 3. descriu el funcionament de les estructures per (for) Compara-les amb les del llenguatge C. Fes especial menció del que a la pàgina indicada anomenen "Enhanced for" o per millorat. L'has vist mai a algun altre llenguatge? A quin i amb quina notació?
