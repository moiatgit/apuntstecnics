Exercici 12. Moviment de responsabilitat

Considera si el mètode correspon realment a la classe on es troba. Mou-lo allà on creguis oportú

El codi resultant hauria de permetre que Client.informe() funcioni amb el següent codi:

 1 public String informe() {
 2     double total = 0;
 3     int bonificacions = 0;
 4     String resultat = "Informe de lloguers del client " +
 5         getNom() +
 6         " (" + getNif() + ")\n";
 7     for (Lloguer lloguer: lloguers) {
 8         bonificacions += lloguer.bonificacions();
 9         // composa els resultats d'aquest lloguer
10         resultat += "\t" +
11             lloguer.getVehicle().getMarca() +
12             " " +
13             lloguer.getVehicle().getModel() + ": " +
14             (lloguer.quantitat() * 30) + "€" + "\n";
15         total += lloguer.quantitat() * 30;
16     }
17 
18     // afegeix informació final
19     resultat += "Import a pagar: " + total + "€\n" +
20         "Punts guanyats: " + bonificacions + "\n";
21     return resultat;
22 }

Nota: ara no serà tan directe com amb quantitat eliminar la variable temporal bonificacions. Paciència.
