Configuració d'Eclipse per C/C++

Entrada ubicada a la categoria eines

Introducció

Aquest document descriu la preparació de Eclipse com a IDE de desenvolupament en C/C++ al sistema operatiu Ubuntu 12.04. Probablement les passes seran similars a altres sistemes operatius, distribucions i versions. En tot cas, pots fer una ullada a la documentació oficial del projecte.

Abans de començar, comprova que no disposes ja de la instal·lació del plugin CDT. Ho pots fer, per exemple, intentant crear un nou projecte C++ a partir de File -> New -> Project.... Si al quadre que apareix hi trobem una secció anomenada C/C++ i hi podem crear un projecte C, podem ignorar la resta d'aquesta guia perquè ja el tenim instal·lat!

Instal·larem el plug-in CDT (C/C++ Development Tooling)

Instal·lació

Realitzarem la instal·lació tot seguint les següents passes.

1. El compilador

Donat que el projecte CDT no ofereix el compilador de C/C++, haurem d'instal·lar un nosaltres. A l'Ubuntu el més probable és que el tinguem ja. Ho podem comprovar amb:

$ g++ --version
g++ (Ubuntu/Linaro 4.6.3-1ubuntu5) 4.6.3
Copyright (C) 2011 Free Software Foundation, Inc.
This is free software; see the source for copying conditions.  There is NO
warranty; not even for MERCHANTABILITY or FITNESS FOR A PARTICULAR PURPOSE.
$ make --version
GNU Make 3.81
Copyright (C) 2006  Free Software Foundation, Inc.
This is free software; see the source for copying conditions.
There is NO warranty; not even for MERCHANTABILITY or FITNESS FOR A
PARTICULAR PURPOSE.

This program built for i686-pc-linux-gnu

En cas d'error amb la primera comanda, caldrà instal·lar el compilador. En cas d'error amb make, comprovarem si gmake sí està disponible. Altrament, haurem d'instal·lar-ho. Donat que aquestes instruccions van més enllà dels objectius d'aquesta petita guia, caldrà cercar les instruccions corresponents pel sistema operatiu objectiu.

2. Instal·lem el plug-in

Naveguem pels menus Help -> Install New Software... i al quadre de diàleg que emergeix premem el botó add. Ens apareix un nou quadre de diàleg amb els camps de text Name i Location. Al primer li posarem CDT i al segon http://download.eclipse.org/tools/cdt/releases/juno

Potser li costarà una estona, però si tot va bé acabarà mostrant dues entrades anomenades CDT Main Features i CDT Optional Features. Desplegarem la primera (fent clic sobre el triangle que apareix abans de cada entrada) i escollirem la primera de les subentrades que hi apareixen: C/C++ Development Tools. Important: no és adequat seleccionar-ho tot! Hi ha dependències que potser faran que no ens funcioni l'aplicació.

Farem Next dos cops i, si estem d'acord amb els termes de la llicència d'ús [1] , sel·leccionem l'opció I accept the terms of license aggreement, premem Finish i esperem una estoneta.

Finalment haurem de reiniciar Eclipse i, en rearrancar, ja tindrem el plug-in disponible.

3. Provem la instal·lació

Naveguem File -> New -> Project... i hi escollim C Project`.

Anomenem el projecte quelcom similar a "HolaCDT" a la localització per defecte i de tipus Empty Project i fem següent.

Al quadre de diàleg Select Configurations premem el botó Advanced settings...

Al quadre de diàleg Properties for HolaCDT naveguem per l'arbre d'opcions C/C++ General-> Code Analysis -> Launching i marquem l'opció Run with build.

Si ens demana que si volem obrir la perspectiva associada a CDT, direm que Sí.

Seleccionem ara crear un nou fitxer font (File -> New -> Source File), li posem com a nom holacdt.c i finalitzem l'ajudant amb el botó Finish.

Escrivim ara el codi del nostre programa després de la capçalera que ens han generat automàticament.

#include<stdio.h>
int main(){
    printf("Hola CDT!\n");
    return 0;
}

Finalment executem el programa amb Run -> Run. Ens demanarà si volem guardar els canvis. Un cop guardats, compilarà i enllaçarà el codi per nosaltres, i finalment ens mostrarà el resultat a la vista anomenada Console que probablement apareixerà sota l'àrea de l'editor. [2]

Anotacions

[1]Si no estem d'acord amb els termes de la llicència, tot i estant en el nostre dret, tenim un problema per continuar seguint aquesta guia.
[2]En cas que aparegui un missatge d'error indicant Launch failed.  Binary not found és que ens hem deixat de configurar l'opció de Run with build a les propietats del projecte. Encara podem arreglar-ho sense haver de començar. Només ens cal indicar-li a Eclipse quin és el main() que ha d'executar. Per fer-ho, guardarem el projecte (File -> Save'' o ``ctrl-s), construïm el projecte amb (Project -> Build all o ctrl-b) i creem la configuració amb Run -> Run configurations...) Al quadre de diàleg que ens apareix, seleccionarem C/C++ Application i si no ens apareix dins el nostre projecte HolaCDT Debug premem el botó de New. Ara, en el nostre projecte premem Search Project... i escollim l'executable holaCDT. Acceptem i premem Run i tot plegat hauria de funcionar ara.
