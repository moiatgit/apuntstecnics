Instal·lació de Maven

Entrada ubicada a la categoria eines amb els tags: util java

Introducció

Aquest breu document t'indica com instal·lar Maven en un sistema GNU/Linux (com ara Debian) sense requerir privilegis d'administració.

Abans de començar

Abans de començar, assegura't que no disposes ja d'una vesió de Maven en el teu sistema.

$ mvn --version
mvn: no s'ha trobat l'ordre

Si el resultat de la comanda és l'indicat abans, voldrà dir que no es troba cap Maven. Si pel contrari, la comanda et mostra alguna informació, llavors ja disposes d'una versió de Maven instal·lada. Per exemple:

$ mvn --version
Apache Maven 3.0.4
Maven home: /usr/share/maven
Java version: 1.7.0_76, vendor: Oracle Corporation
Java home: /usr/lib/jvm/java-7-oracle/jre
Default locale: ca_ES, platform encoding: UTF-8
OS name: "linux", version: "3.5.0-54-generic", arch: "amd64", family: "unix"

En aquest cas, hauràs de decidir si vols quedar-te amb aquesta versió o bé continuar amb aquesta guia per obtenir una de més moderna.

Instruccions

Per començar a instal·lar Maven en el teu sistema, primer de tot, descarrega la darrera versió de Maven del lloc de descàrregues d'Apache

Si no tens una preferència clara de com i on fer-ho, et proposo les següents passes [1]:

  1. Descarrega el paquet amb la darrera versió de Maven:

    $ cd /tmp
$ wget -c http://apache.rediris.es/maven/maven-3/3.3.1/binaries/apache-maven-3.3.1-bin.tar.gz
$ wget -c http://www.apache.org/dist/maven/maven-3/3.3.1/binaries/apache-maven-3.3.1-bin.tar.gz.md5
$ md5sum apache-maven-3.3.1-bin.tar.gz | cut -d\  -f 1 | diff -b - apache-maven-3.3.1-bin.tar.gz.md5

    Si la darrera comanda ha generat algun missatge, voldrà dir que has tingut algun problema en la descàrrega del fitxer. Torna a descarregar-t'ho i intenta-ho de nou.

  2. A continuació descomprimirem Maven en un directori que sigui accessible per la variable PATH.

    $ mkdir ~/soft/apache-maven
$ cd ~/soft/apache-maven
$ tar -xzvf /tmp/apache-maven-3.3.1-bin.tar.gz
$ ln -s apache-maven-3.3.1 actual

  3. Ara caldrà afegir Maven a la teva variable PATH. Et proposo que ho facis al fitxer ~/.bashrc

    Així doncs, edita el fitxer ~/.bashrc i afegeix al final d'aquest la següent línia [2]:

    PATH=$PATH:~/soft/apache-maven/actual/bin

    Finalment guarda els canvis.

  1. Comprovem que tot ha anat bé

    Obre una nova consola i escriu

    $ mvn --version
Apache Maven 3.3.1 (cab6659f9874fa96462afef40fcf6bc033d58c1c; 2015-03-13T21:10:27+01:00)
Maven home: /home/moi/soft/apache-maven/actual
Java version: 1.8.0_40, vendor: Oracle Corporation
Java home: /mnt/dades/soft/jdk/jdk1.8.0_40/jre
Default locale: ca_ES, platform encoding: UTF-8
OS name: "linux", version: "3.2.0-4-amd64", arch: "amd64", family: "unix"

Si tot ha anat bé, enhorabona, ja tens el teu flamant Maven instal·lat només pel teu usuari sense necessitat de privilegis d'administració.

Notes

[1]Potser voldràs assegurar-te que la versió més actual continua sent la que et proposo. Altrament, pots canviar les instruccions per la darrera versió.
[2]En cas que ja estigués instal·lat el Maven al teu sistema, però t'interessi aquesta darrera versió, és possible que calgui posar el path al nou Maven abans que la resta de camins de PATH
